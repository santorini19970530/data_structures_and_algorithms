Eucid's Algorithm / Euclidean Algorithm

* Greatest Common Divisor of a and b

```
// version 1
function GreatestCommmonDivisor(a, b)
  for 0 <= i <= a + b do
    if a = b then
      break
    end if
    if a > b then
      a = a - b
    else
      b = b - a
    end if
  end for
  return a
end function
```

```
// version 2
function GreatestCommmonDivisor(a, b)
  while a != b do
    if a > b then
      a = a - b
    else
      b = b - a
    end if
  end while
  return a
end function
```
